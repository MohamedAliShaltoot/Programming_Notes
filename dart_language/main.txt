# Dart Language Notes :

1) Casting in Dart

  1- if number is var x =10;
     var y = x.toDouble();
     var y = x.toString();
     and so on

  2- but if was a string like var x ="10";
     var y = num.parse(x);   

 2) Checking the values in Dart    

  1- if value is var x ="ali";
     var y = x ?? "Guest"; // here y = ali

     but if 
     var x =null;
     var y = x ?? "Guest"; // here y = Guest
